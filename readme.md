## Simple example ##
This project propose a very simple example for c++ developer using GitHub workflows to run Google tests on Windows and Ubuntu platforms.

Local configuration used:
* Windows 10
* Visual code 1.42.1
  * CMake 0.0.17
  * CMake Tools 1.3.1
* CMake 3.16.4
* MinGW GCC-8.2.0-5
* Googletest 23b2a3b1
